# security-robot-dog

## Project Overview
The Security Robot Dog is a conceptual 3D model designed using Tinkercad. It represents a robotic dog developed for security and surveillance purposes. The model features a simple four-legged structure with a front-mounted camera and built-in Wi-Fi connectivity. The design demonstrates the basic mechanical structure of a security robot while maintaining a simple and lightweight appearance.

## Project Objectives
- Design a four-legged security robot using Tinkercad.
- Practice 3D modeling and mechanical design.
- Create a stable robot body with leg joints.
- Add a front camera for surveillance.
- Represent wireless communication using a built-in Wi-Fi module.

## Features
- Four-legged robotic design.
- Front-mounted security camera.
- Built-in Wi-Fi connectivity.
- Lightweight and compact structure.
- Simple mechanical design suitable for educational purposes.

## Main Components
- Robot body
- Four legs
- Leg joints
- Front camera
- Built-in Wi-Fi module

## Motor Selection
The robot is designed to use Servo Motors for each leg joint. Servo motors provide precise angle control, making them suitable for robotic leg movement, stability, and smooth walking.

## Software Used
- Tinkercad
